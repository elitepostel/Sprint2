Этот проект сравнивает генерацию текста с помощью LSTM и DistilGPT-2.  
Код включает обучение, оценку (ROUGE) и сохранение весов моделей.

## Структура проекта

- `solution.py` — основной файл запуска

- `src/` — исходный код

  - `lstm_model.py` # код lstm модели

  - `lstm_train.py` # код обучения модели

  - `data_utils.py` # Обработка датасета

  - `eval_transformer.py` # код с запуском и замером качества трансформера


## Запуск

```bash

python solution.py
